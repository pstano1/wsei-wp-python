# wsei-wp (python)

Implements following patterns:

- MVC
- Observer

### Usage

- pasue/resume - `space bar` (paused game is slightly dimmed)
- save - `cmd + s`
- load - `cmd + l`

### Running locally

```console
git clone https://github.com/pstano1/wsei-wp-python.git
cd wsei-wp-python
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```
